# ResellHub – Second-Hand Marketplace Platform

## Project Purpose
ResellHub is an online marketplace where users can buy and sell pre-owned products safely and efficiently. Sellers list items they no longer need, buyers browse and purchase securely, and admins moderate the platform — helping reduce waste and promote affordable, sustainable shopping.

## Live URL
https://resell-sable.vercel.app

## Key Features
- Role-based dashboards for Buyer, Seller, and Admin with protected routes
- Email/password & Google authentication (Better Auth) with role selection
- Add, edit, and delete product listings with real image uploads (ImgBB)
- Search, filter, sort, and paginate products
- Wishlist for saving favorite products
- Secure Stripe checkout and payment processing
- Order lifecycle tracking (Pending → Accepted → Processing → Shipped → Delivered)
- Admin moderation: approve/reject products, manage and block/unblock users
- JWT-secured private APIs with role-based authorization
- Animated, dynamic homepage with live marketplace statistics (Framer Motion)
- Fully responsive, dark-themed UI with custom 404 page

## NPM Packages Used
**Frontend:** next, react, better-auth, @heroui/react, @gravity-ui/icons, react-hot-toast, framer-motion, recharts, tailwindcss

**Backend:** express, cors, dotenv, mongodb, stripe, jose
